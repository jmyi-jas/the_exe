# 法院工具箱 Court Document Toolkit

> 自动化处理法院文书的桌面工具，支持文书下载、起诉状提取、裁判文书初筛分类。
> **本工具面向零编程基础用户，请按以下步骤操作。**

---

## 第一步：安装 Python

1. 打开浏览器，访问 [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. 点击黄色按钮 **Download Python 3.x.x** 下载安装包
3. 运行安装包，**务必勾选底部 `Add Python to PATH`** 选项，再点击 Install Now

> ✅ 验证安装：按 `Win + R`，输入 `cmd` 回车，在黑窗口中输入 `python --version`，出现版本号即为成功

---

## 第二步：安装依赖库

按 `Win + R`，输入 `cmd` 回车，将以下命令**完整复制粘贴**到黑窗口中，回车执行：

**必需（所有功能都需要）：**
```
pip install pandas requests openpyxl pdfplumber python-docx
```

**可选（扫描版 PDF 才需要，普通用户可跳过）：**
```
pip install pymupdf easyocr numpy
```

> ⚠️ 安装过程中若出现 `error` 字样，可尝试在命令前加 `pip install -i https://pypi.tuna.tsinghua.edu.cn/simple` 换用国内镜像源

---

## 第三步：运行程序

找到下载好的 `court_toolkit.py` 文件，**双击**即可打开。

若双击无效，可在该文件所在文件夹内，按住 `Shift` + 鼠标右键，选择「在此处打开命令窗口」，输入：
```
python court_toolkit.py
```

---

## 功能使用说明

程序启动后，顶部有三个选项卡，按需选择：

---

### 📥 文书下载

**用途：** 批量从法院短信链接下载电子文书，自动按案号归档

**使用步骤：**
1. 将包含法院短信的记录整理成 Excel 文件（`.xlsx`）
2. 点击「浏览」选择该 Excel 文件
3. 填写短信内容所在的列名（默认 `Message Body`）和时间列名（默认 `Date`）
4. 选择文件下载到哪个文件夹
5. 点击「▶ 开始执行」

**输出结果：** 每条短信对应一个以案号命名的文件夹，内含下载的文书文件和短信备份

---

### 📄 起诉状提取

**用途：** 批量读取起诉状文件，自动提取案号、原被告等信息，汇总为 Excel

**使用步骤：**
1. 将所有案件文件夹（每个文件夹内含起诉状）放在同一个根目录下
2. 点击「浏览」选择该根目录
3. 点击「▶ 开始提取」

**输出结果：**
- 根目录下生成 `起诉状信息汇总.xlsx`，包含所有案件的案号、原告、被告、诉讼请求等
- 各案件文件夹自动重命名为「原告名称 + 原案号」格式

> ⚠️ 若起诉状为扫描版 PDF，需提前安装可选依赖库，首次使用 OCR 功能需联网下载模型（约 100MB），加载耗时 10–30 秒，请耐心等待

---

### 🔍 初筛裁判书

**用途：** 自动识别文件夹内的裁判文书类型，为文件夹添加分类前缀

**使用步骤：**
1. 点击「浏览」选择包含多个案件子文件夹的根目录
2. 点击「▶ 开始初筛」

**输出结果：** 自动为文件夹名添加以下前缀之一：

| 前缀 | 识别依据 | 优先级 |
|------|---------|--------|
| `【传票】` | 文件夹内含"传票"PDF | 最高 |
| `【裁定书】` | 文件夹内含"裁定书"或"本院法律文书正本_1"PDF | 第二 |
| `【判决书】` | 文件夹内含"判决书"PDF | 第三 |
| `【调解书】` | 文件夹内含"调解书"PDF | 最低 |

> 已有前缀的文件夹自动跳过，不会重复处理

## 常见问题

**Q：双击 py 文件一闪而过，没有窗口弹出？**
> 用命令行方式运行（见第三步），查看错误信息，通常是某个依赖库未安装

**Q：提示"找不到列：Message Body"？**
> 打开你的 Excel，查看第一行表头的实际列名，将程序中的列名输入框改为对应名称

**Q：Excel 文件保存失败？**
> 请先关闭已用 Excel 打开的同名文件，再重新执行

**Q：OCR 识别效果很差？**
> 扫描版 PDF 质量过低时识别率会下降，建议使用分辨率较高的扫描件

---

## 注意事项

- 程序会**直接修改文件夹名**，操作前建议备份数据
- 所有操作均在本地进行，不会上传你的文件到任何服务器
- 本工具仅供提升工作效率使用，请确保使用方式合法合规
